##🏥 HealthChain: Decentralized EHR System
HealthChain is a patient-centric, decentralized Electronic Health Record (EHR) platform. It leverages blockchain technology to return data ownership to patients, ensuring medical records are immutable, secure, and accessible only to authorized healthcare providers.

🚀 Live Demo: https://healthchain-frontend.vercel.app

##🌟 Key Features


Absolute Patient Ownership: Patients have the sole authority to grant or revoke access to their medical records for doctors and diagnostic centers.

Immutable Audit Trail: All access permissions and record additions are anchored to the Ethereum Sepolia blockchain, creating a permanent, tamper-proof history.

Decentralized Storage: Medical files are stored on IPFS (via Pinata), ensuring no single point of failure and high data availability.

Hybrid Architecture: Combines the speed of MongoDB Atlas for metadata with the security of Blockchain for integrity.

Real-time Feedback: Integrated Transaction Stepper provides users with live status updates during blockchain mining and IPFS pinning.
